Параллельный текст «Манифеста Коммунистической партии»

04.11.2025

Параллельный текст «Манифеста Коммунистической партии» основан на издании «В. Ф. Шелике и Г. И. Шелике, Новая редакция перевода на русский язык “Манифеста Коммунистической партии” К. Маркса и Ф. Энгельса и материалы к нему».

Сайт «Манифеста» воплощает несколько текстологических идей: хранение исходных данных в простом текстовом формате, открытый доступ к ним и разделение данных и их визуализации.

Тексты хранятся в виде отдельных файлов с использованием простой разметки Markdown. Их можно читать в исходном виде на любом устройстве.

Исходные тексты находятся в открытом доступе. Каждый желающий может использовать их по своему усмотрению — например, создать собственную визуализацию параллельного текста. Чаще всего самая трудоёмкая часть оцифровки — вычитка текста. Подлинное обобществление текста предполагает свободный доступ без необходимости повторного выполнения этой работы.

Визуальная часть почти полностью создана с помощью нейросетей через Cursor. Подробная техническая информация доступна в файле readme.md.

На десктопных устройствах тексты отображаются в соседних колонках.

В мобильной версии абзацы на разных языках идут друг за другом.

Разделение исходных данных и визуализации позволяет легко изменить или добавить новый способ отображения параллельного текста. Так же просто можно добавить переводы на другие языки.

Сайт состоит из одного HTML-файла, который заранее генерируется на сервере с помощью GitHub Actions. Вся техническая часть доступна в открытом репозитории и может использоваться для любых других текстов.

Для развития сайта «Параллельный текст Манифеста» будет полезна помощь в следующих направлениях:

  • проверка корректности текстов, выделений и примечаний (исходный PDF);
  • добавление переводов на другие языки;
  • доработка или создание новой визуальной реализации.
