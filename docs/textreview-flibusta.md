## Описание

[Флибуста](https://flibusta.is) — онлайн-библиотека, основанная на формате FB2.

> 1\) Флибуста - независимая библиотека электронных книг вместе с образовавшимся вокруг нее сообществом.
>
> 2\) Флибуста основана на идее свободного доступа к книгам. Все материалы доступны каждому читателю бесплатно и безвозмездно.
>
> 3\) Библиотека устроена по принципу вики. Это означает, что каждый пользователь может непосредственно участвовать в жизни библиотеки, загружая и обрабатывая книги. Наполнение и упорядочивание книжных фондов производится целиком пользователями, администрация ими не занимается. Запрещено использовать искусственный интеллект.
> 
> 6\) Флибуста - библиотека художественная. Естественно-научную и техническую литературу следует заливать в дружественную библиотеку Либген ([Источник](https://flibusta.is/node/4023))

У сайта есть зеркала [в TOR](http://flibustaongezhld6dibs2dps6vm4nvqg2kp7vgowbu76tzopgnhazqd.onion) и [в I2P](http://flibusta.i2p). Версия без доступа к текстам: [flisland.net](http://flisland.net). Дополнительно у библиотеки есть домен в EmerCoin через систему OpenNIC.

На случай проблем с основными адресами существуют альтернативные способы оповещения: [x.com/flibusta](https://x.com/flibusta) и [librusec.ucoz.de](http://librusec.ucoz.de/forum/26).

> 5\) По техническим и организационным причинам ряд возможностей доступен только зарегистрированным пользователям. Регистрация свободная.
>
> 6\) Некоторые функции работы с библиотекой доступны не всем пользователям, как правило по причине возможного злоупотребления. Желающий произвести какое-либо недоступное ему действие может запросить администрацию или просто отписаться на форуме.

## Монетизация

На сайте библиотеки реклама отсутствует.

> 4) Флибуста финансируется только ее создателями и через добровольные пожертвования. В частности сайт не содержит платной рекламы. ([Источник](https://flibusta.is/node/4023))

Библиотека разрешает авторам книг размещать ссылки на свои страницы с информацией о способах поддержки.

> В: А могу я дать в библиотеке свои электронные кошельки?
О: Дайте их на подконтрольной только вам страничке, а здесь - ссылку на оную. Желающий заплатить один лишний раз кликнет, но будет уверен, что кошельки правильные.
И вот ещё: верифицируйтесь, пожалуйста, в этой теме. Не все ведь знают точно что ваша страничка - именно ваша, а не чья-то... ([Источник](https://flibusta.is/node/4023))

## Технологии

> 7) Главный и основной формат библиотеки - fb2. Заливать производные от него форматы желательно только при наличии веской причины. ([Источник](https://flibusta.is/node/4023))

У сайта есть [OPDS-каталог](https://flibusta.is/opds).

На сайте присутствует инструмент сравнения книг, вероятно, для помощи в устранении дублей. Однако воспользоваться им не удалось.

![](/images/textreview/flibusta/flibusta_17.png)

Сайт работает на [Drupal](https://www.drupal.org/) или его форке [Pressflow](https://www.pressflow.org).

Используется база данных MySQL. Точная информация из дампа:

```
MySQL dump 10.14  Distrib 5.5.68-MariaDB, for Linux (x86_64)
```

На [отдельной странице](https://flibusta.is/sql/) можно скачать актуальную копию базы данных. Пример одной таблицы:

```sql
CREATE TABLE `libavtor` (
  `BookId` int(10) unsigned NOT NULL DEFAULT '0',
  `AvtorId` int(10) unsigned NOT NULL DEFAULT '0',
  `Pos` tinyint(4) unsigned NOT NULL DEFAULT '0',
  PRIMARY KEY (`BookId`,`AvtorId`),
  KEY `iav` (`AvtorId`)
) ENGINE=MyISAM DEFAULT CHARSET=utf8 COLLATE=utf8_unicode_ci;
```

![Страница с файлами базы данных.](/images/textreview/flibusta/flibusta_18.png)

Всю библиотеку можно скачать с трекера [booktracker.org](https://booktracker.org). Там же доступны ежемесячные обновления:

![](/images/textreview/flibusta/flibusta_19.png)

С помощью этих копий энтузиасты создают новые зеркала, что делает архив книг трудноблокируемым и неуничтожимым.

## Заключение

Основное отличие Флибусты от КулЛиб — политика в отношении авторского права. Одна библиотека размещает любые книги и прилагает усилия, чтобы они оставались доступны (например, с помощью полного архива на торрентах), а другая удаляет книги по запросу правообладателей.

Техническая сторона сайта Флибусты выглядит менее развитой, чем у КулЛиб. Сообщество, судя по комментариям, блогам и форуму, кажется более живым у Флибусты.

Особый интерес представляют меры, которые предпринимает Флибуста для обеспечения своей доступности.
