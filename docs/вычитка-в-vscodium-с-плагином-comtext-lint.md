# Вычитка в VSCodium с плагином Comtext-lint

**07.03.2026**

[VSCodium](https://vscodium.com) — удобный свободный текстовый редактор. Ниже описаны функции, которые помогут при вычитке текста.

## Установка плагина Comtext-lint

В левой панели нажмите на значок расширений (1), в поиске введите `comtext` (2), выберите плагин `Comtext-lint` (3) и нажмите `Install` (4).

![](/images/вычитка-в-vscodium-с-плагином-comtext-lint/вычитка-в-vscodium-с-плагином-comtext-lint-01.png)

## Проверка формата

Проверка формата работает автоматически. Результаты отображаются в панели `Problems`.

![](/images/вычитка-в-vscodium-с-плагином-comtext-lint/вычитка-в-vscodium-с-плагином-comtext-lint-02.png)

Проверка срабатывает при сохранении файла в формате Комтекст с расширением `.md` или `.ct`. Важно, чтобы во фронтматтере было указано свойство `format: comtext`.

Если в файле есть ошибка, она отобразится в списке проблем (1), а файл будет подсвечен в дереве проекта (2).

![](/images/вычитка-в-vscodium-с-плагином-comtext-lint/вычитка-в-vscodium-с-плагином-comtext-lint-03.png)

## Исправление опечаток после OCR

Плагин содержит функцию [исправления частых ошибок после OCR](https://github.com/comtextspace/ocr-fix). Применить её можно через контекстное меню редактора или для всего каталога.

![](/images/вычитка-в-vscodium-с-плагином-comtext-lint/вычитка-в-vscodium-с-плагином-comtext-lint-04.png)

![](/images/вычитка-в-vscodium-с-плагином-comtext-lint/вычитка-в-vscodium-с-плагином-comtext-lint-05.png)

## Просмотр изменений

После автоматического обновления файла проверьте изменения. Если вы используете Git для хранения файлов, VSCodium покажет их в специальном окне.

В панели инструментов выберите пункт для контроля версий (1) и нужный файл (2).

![](/images/вычитка-в-vscodium-с-плагином-comtext-lint/вычитка-в-vscodium-с-плагином-comtext-lint-06.png)

Кнопки в правой верхней панели редактора позволяют перемещаться между изменениями. Это удобно для больших файлов с несколькими правками.

![](/images/вычитка-в-vscodium-с-плагином-comtext-lint/вычитка-в-vscodium-с-плагином-comtext-lint-07.png)
