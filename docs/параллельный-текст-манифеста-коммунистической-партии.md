# Параллельный текст «Манифеста Коммунистической партии»



**04.11.2025**

[Параллельный текст «Манифеста Коммунистической партии»](https://the-communist-manifesto.comtext.space) основан на издании «[В. Ф. Шелике](https://ru.wikipedia.org/wiki/Шелике,_Вальтраут_Фрицевна) и Г. И. Шелике, Новая редакция перевода на русский язык “Манифеста Коммунистической партии” К. Маркса и Ф. Энгельса и материалы к нему».

Сайт «Манифеста» воплощает несколько текстологических идей: хранение исходных данных в простом текстовом формате, открытый доступ к ним и разделение данных и их визуализации.

Тексты хранятся [в виде отдельных файлов](https://github.com/comtextspace/the-communist-manifesto/tree/main/data) с использованием простой разметки [Markdown](https://ru.wikipedia.org/wiki/Markdown). Их можно читать в исходном виде на любом устройстве.

Исходные тексты находятся в открытом доступе. Каждый желающий может использовать их по своему усмотрению — например, создать собственную визуализацию параллельного текста. Чаще всего самая трудоёмкая часть оцифровки — вычитка текста. Подлинное обобществление текста предполагает свободный доступ без необходимости повторного выполнения этой работы.

Визуальная часть почти полностью создана с помощью нейросетей через [Cursor](https://cursor.com). Подробная техническая информация доступна в файле [readme.md](https://github.com/comtextspace/the-communist-manifesto/blob/main/readme.md).

На десктопных устройствах тексты отображаются в соседних колонках.

![](/images/параллельный-текст-манифеста-коммунистической-партии-1.png)

В мобильной версии абзацы на разных языках идут друг за другом.

![](/images/параллельный-текст-манифеста-коммунистической-партии-2.png)

Разделение исходных данных и визуализации позволяет легко изменить или добавить новый способ отображения параллельного текста. Так же просто можно добавить переводы на другие языки.

Сайт состоит из одного HTML-файла, который заранее генерируется на сервере с помощью [GitHub Actions](https://github.com/features/actions). Вся техническая часть доступна [в открытом репозитории](https://github.com/comtextspace/the-communist-manifesto) и может использоваться для любых других текстов.

Для развития сайта «Параллельный текст Манифеста» будет полезна помощь в следующих направлениях:

* проверка корректности текстов, выделений и примечаний ([исходный PDF](https://vk.com/wall-81495923_14503));
* добавление переводов на другие языки;
* доработка или создание новой визуальной реализации.
